blog sederhana

contoh blog sederhana dengan box model

Bok model adalah model yang dipakai untuk menampilkan elemen html di browser. model ini akan memudahkan kita mengatur ukuran,posisi, dan properti lainnya

Dengan memahami box model,kita akan mudah mengatur space pada elemen. ini berguna dalam membuat layout supaya terlihat lebih rapi dan estetik.
